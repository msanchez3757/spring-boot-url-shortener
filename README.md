Spring Boot URL Shortener

A full-stack URL shortener built with Spring Boot and Thymeleaf. It supports authentication, role-based access control, and administrative moderation.

Core Features:
Guests
Create and use short links without an account
Registered Users
Register and log in
Create short links with
Custom expiration dates
Private visibility
Manage their links in a personal dashboard
Admins
View all links
Delete expired or inappropriate links

Backend:
Spring Boot
Spring Security
Spring Data JPA
Flyway
PostgreSQL

Frontend:
Thymeleaf
HTML, CSS, JavaScript

Deployment:
Docker
Spring Boot Buildpacks
Built to practice:

Full-stack development with Spring Boot

Authentication and authorization

Database design with JPA and Flyway

Docker-based deployment
